# php-collections
A PHP library that defines collections of objects.

*Project under development.*

## Installation

composer require antoniokadid/php-collections

## Requirements
* PHP 7.1
* JSON Extension

## Features

Collections are created with method chaining in mind.

#### Collections
* ArrayList
  * Sorting
  * Grouping
* Map
* Queue
* Stack
